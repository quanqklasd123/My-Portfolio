# My-Portfolio

Premium portfolio website for a Full-stack Developer Intern built with:

- React + Vite
- Tailwind CSS
- Framer Motion
- React Router
- `shadcn/ui`-style reusable components

## Run locally

```bash
npm install
npm run dev
```

## Structure

```text
src/
  components/
    layout/
    sections/
    ui/
  hooks/
  lib/
  pages/
```

Update the sample personal data in `src/lib/data.ts` to match your real profile, projects, and links.
